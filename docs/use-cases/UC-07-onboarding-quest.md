# UC-07 — Quest Onboarding & Ưu đãi Người mới

Dẫn dắt user mới trải nghiệm sản phẩm qua chuỗi nhiệm vụ, thưởng voucher.

| # | Tính năng | Mô tả |
|---|---|---|
| 1 | Xem tiến trình quest | User xem tiến độ hoàn thành các nhiệm vụ onboarding |
| 2 | Hoàn thành một quest | Đánh dấu một nhiệm vụ onboarding đã hoàn thành (không lặp lại) |
| 3 | Nhận voucher người mới | User hoàn thành toàn bộ quest, nhận voucher giảm giá 50% dành cho người mới |
