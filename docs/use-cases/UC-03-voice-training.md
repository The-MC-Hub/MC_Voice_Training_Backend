# UC-03 — Luyện giọng AI (Voice Training)

Luồng chính của sản phẩm: luyện đọc kịch bản, AI chấm điểm giọng nói, xem lịch sử luyện tập.

| # | Tính năng | Mô tả |
|---|---|---|
| 1 | Duyệt thư viện bài luyện | User xem/tìm kiếm bài luyện giọng, lọc theo chuyên mục (Wedding, Talkshow, Event, Podcast...) |
| 2 | Xem chi tiết bài luyện | User xem nội dung kịch bản, độ khó, WPM mục tiêu của một bài luyện |
| 3 | Bài luyện nổi bật | Hệ thống gợi ý một số bài luyện nổi bật cho user |
| 4 | Phân tích giọng nói AI | User ghi âm đọc kịch bản, AI chấm điểm phát âm, nhịp điệu, tốc độ nói |
| 5 | Luyện tập thử cho khách (guest) | Người chưa đăng nhập được dùng thử 1 lần phân tích giọng miễn phí, giới hạn theo IP/cooldown |
| 6 | Proxy phân tích giọng trực tiếp | User gửi audio phân tích trực tiếp qua AI service (không qua hàng đợi) |
| 7 | Xem lịch sử luyện tập | User xem lại toàn bộ các phiên luyện tập đã thực hiện |
| 8 | Xem chi tiết một phiên luyện tập | User xem kết quả chi tiết (điểm số, feedback) của một phiên cụ thể |
| 9 | Thống kê độ khó thích ứng | Xem thống kê điều chỉnh độ khó của một bài luyện dựa trên dữ liệu luyện tập cộng đồng |
| 10 | Tạo giọng đọc mẫu (TTS) | Hệ thống tạo file âm thanh mẫu đọc kịch bản để user tham khảo |
| 11 | Xem thời gian cooldown dùng thử | Xem thời gian chờ giữa các lượt dùng thử miễn phí cho khách |
| 12 | Admin thêm bài luyện | Admin tạo bài luyện giọng mới (kịch bản, tiêu chí chấm điểm, WPM mục tiêu) |
| 13 | Admin sửa bài luyện | Admin chỉnh sửa nội dung/thông tin bài luyện đã có |
| 14 | Admin xoá bài luyện | Admin xoá bài luyện khỏi hệ thống |
