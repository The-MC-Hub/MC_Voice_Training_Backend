# UC-04 — Khoá học & Lộ trình học (Courses & Learning Path)

Khoá học có cấu trúc gồm bài luyện giọng + bài đọc lý thuyết + quiz + chứng chỉ hoàn thành.

| # | Tính năng | Mô tả |
|---|---|---|
| 1 | Xem lộ trình học (roadmap) | User xem các khoá học milestone sắp xếp theo cấp độ từ cơ bản đến nâng cao |
| 2 | Xem loại khoá học | User xem danh sách các loại khoá học (Wedding, Corporate, Talkshow, Speaking...) |
| 3 | Danh sách khoá học | User xem toàn bộ khoá học đang hoạt động, lọc theo loại |
| 4 | Xem chi tiết khoá học | User xem nội dung, bài luyện, bài đọc, quiz của một khoá học cụ thể |
| 5 | Xem bài đọc lý thuyết | User đọc nội dung một bài đọc lý thuyết (reading guide) |
| 6 | Đăng ký khoá học | User đăng ký tham gia một khoá học (theo gói cước hoặc mua lẻ) |
| 7 | Nhận khoá học tặng | User được tặng miễn phí một khoá học (quà chào mừng, bỏ qua kiểm tra gói cước) |
| 8 | Đánh dấu hoàn thành bài luyện | User đánh dấu đã hoàn thành một bài luyện trong khoá học |
| 9 | Đánh dấu hoàn thành bài đọc | User đánh dấu đã hoàn thành một bài đọc trong khoá học |
| 10 | Nộp bài quiz cuối khoá | User làm và nộp bài trắc nghiệm cuối khoá để nhận chứng chỉ |
| 11 | Xem khoá học đã đăng ký | User xem danh sách khoá học mình đang tham gia |
| 12 | Xem chứng chỉ đã đạt được | User xem danh sách chứng chỉ hoàn thành khoá học |
| 13 | Ghi chú/highlight bài đọc | User tạo, sửa, xoá highlight và ghi chú trên nội dung bài đọc |
| 14 | Admin quản lý khoá học | Admin xem, tạo, sửa, xoá khoá học và xem thống kê đăng ký |
| 15 | Admin chỉnh giá khoá học | Admin thiết lập giá và giảm giá cho khoá học mua lẻ |
