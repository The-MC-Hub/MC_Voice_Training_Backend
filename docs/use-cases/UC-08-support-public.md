# UC-08 — Trang công khai, Hỗ trợ & Báo cáo

Trang landing công khai, liên hệ hỗ trợ, báo cáo vi phạm, upload media.

| # | Tính năng | Mô tả |
|---|---|---|
| 1 | Xem trang landing công khai | Khách vãng lai xem dữ liệu trang giới thiệu sản phẩm |
| 2 | Xem thống kê luyện tập nổi bật | Khách xem số liệu tổng quan luyện tập để tăng độ tin cậy (social proof) |
| 3 | Xem danh sách enum công khai | Frontend lấy danh sách các loại vai trò user, lý do báo cáo để hiển thị dropdown |
| 4 | Gửi liên hệ/hỗ trợ | Khách gửi form liên hệ, nội dung được email đến hộp thư hỗ trợ |
| 5 | Upload file media | User upload file (ảnh chat, tài liệu) lên hệ thống lưu trữ, nhận về URL |
| 6 | Gửi báo cáo vi phạm | User gửi báo cáo/khiếu nại về nội dung hoặc người dùng khác |
| 7 | Xem báo cáo của tôi | User xem lại các báo cáo mình đã gửi |
| 8 | Admin xem danh sách báo cáo | Admin xem toàn bộ báo cáo, lọc theo trạng thái đang chờ xử lý |
| 9 | Admin xử lý báo cáo | Admin cập nhật trạng thái và ghi chú xử lý cho một báo cáo |
