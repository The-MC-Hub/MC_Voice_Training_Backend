# UC-05 — Cộng đồng & Bảng xếp hạng (Community & Leaderboard)

Tính năng gắn kết cộng đồng: xếp hạng, giải đấu, đăng bài mạng xã hội.

| # | Tính năng | Mô tả |
|---|---|---|
| 1 | Xem thống kê cộng đồng | User xem số liệu tổng quan hoạt động cộng đồng (tổng user, tổng phiên luyện...) |
| 2 | Xem bảng xếp hạng | User xem bảng xếp hạng theo tiêu chí (streak, chăm chỉ, độ chính xác, số phiên) và theo kỳ (tuần/tháng) |
| 3 | Xem thứ hạng của bản thân | User xem vị trí xếp hạng hiện tại của chính mình |
| 4 | Xem giải đấu đang diễn ra | User xem danh sách các "arena" thi đấu cộng đồng đang hoạt động |
| 5 | Admin tạo giải đấu | Admin tạo một giải đấu/thử thách cộng đồng mới |
| 6 | Admin sửa giải đấu | Admin chỉnh sửa thông tin giải đấu đã tạo |
| 7 | Admin xoá giải đấu | Admin xoá một giải đấu |
| 8 | Xem bài đăng mạng xã hội | User xem các bài đăng (fanpage) đang được hiển thị công khai |
| 9 | Ghi nhận click bài đăng | Hệ thống ghi nhận lượt click vào bài đăng mạng xã hội để phân tích |
| 10 | Admin quản lý bài đăng mạng xã hội | Admin tạo, sửa, xoá, bật/tắt hiển thị bài đăng mạng xã hội |
