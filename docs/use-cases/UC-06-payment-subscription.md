# UC-06 — Thanh toán & Gói cước (Payment & Subscription)

Quản lý gói cước, thanh toán qua PayOS, mã giảm giá.

| # | Tính năng | Mô tả |
|---|---|---|
| 1 | Xem danh sách gói cước | User xem các gói cước đang mở bán và giá tương ứng (FREE/DAILY/BASIC/FULL/ANNUAL) |
| 2 | Xem ưu đãi giới hạn thời gian | User xem các flash-deal giảm giá đang áp dụng |
| 3 | Áp dụng mã giảm giá | User nhập mã giảm giá để xem trước giá cuối cùng trước khi thanh toán |
| 4 | Tạo đơn thanh toán gói cước | User tạo link thanh toán PayOS để mua/nâng cấp gói cước |
| 5 | Tạo đơn thanh toán khoá học lẻ | User tạo link thanh toán PayOS để mua riêng một khoá học |
| 6 | Webhook xác nhận thanh toán | Hệ thống nhận webhook từ PayOS để xác nhận giao dịch hoàn tất |
| 7 | Xem trạng thái thanh toán | User xem trạng thái gói cước hiện tại và lịch sử giao dịch |
| 8 | Admin giả lập thanh toán thành công | Admin giả lập một giao dịch thành công để phục vụ test/dev |
| 9 | Admin xử lý thủ công giao dịch | Admin đánh dấu hoàn tất và kích hoạt gói cước cho một giao dịch đang chờ |
| 10 | Admin quản lý gói cước | Admin xem/sửa cấu hình các gói cước (giá, thời hạn, giới hạn AI session) |
| 11 | Admin khởi tạo gói Ngày (DAILY) | Admin seed/tạo gói cước theo ngày mặc định lần đầu |
| 12 | Admin quản lý mã giảm giá | Admin xem, tạo, sửa, xoá mã giảm giá |
| 13 | Xem voucher của tôi | User xem toàn bộ voucher đang có (đã dùng và chưa dùng) |
| 14 | Xem voucher khả dụng | User xem các voucher còn hiệu lực và chưa sử dụng |
