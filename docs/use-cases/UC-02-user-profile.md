# UC-02 — Hồ sơ Người dùng & MC (User & MC Profile)

Quản lý hồ sơ cá nhân, streak luyện tập, hồ sơ MC chuyên nghiệp và chứng chỉ.

| # | Tính năng | Mô tả |
|---|---|---|
| 1 | Xem chuỗi đăng nhập (streak) | User xem số ngày luyện tập liên tiếp, khung avatar theo cấp độ, số lần đóng băng streak còn lại |
| 2 | Đóng băng streak | User dùng lượt đóng băng để bảo toàn streak khi bỏ lỡ một ngày luyện tập |
| 3 | Xem dashboard MC | MC xem thống kê tổng quan hoạt động luyện tập, tiến trình của bản thân |
| 4 | Cập nhật hồ sơ MC | MC chỉnh sửa thông tin hồ sơ chuyên nghiệp (giới thiệu, chuyên môn, kinh nghiệm) |
| 5 | Thêm chứng chỉ | MC thêm chứng chỉ/bằng cấp chuyên môn vào hồ sơ |
| 6 | Xem chứng chỉ theo hồ sơ MC | Xem danh sách toàn bộ chứng chỉ của một MC cụ thể |
| 7 | Admin duyệt chứng chỉ | Admin xác minh (duyệt) chứng chỉ do MC gửi lên |
| 8 | Xoá chứng chỉ | MC xoá chứng chỉ của chính mình khỏi hồ sơ |
| 9 | Khám phá hồ sơ MC công khai | Người dùng công khai tìm kiếm/lọc hồ sơ MC theo chuyên mục |
| 10 | Xem hồ sơ MC công khai chi tiết | Xem chi tiết một hồ sơ MC cụ thể trên trang public |
